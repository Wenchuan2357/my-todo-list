# 个人待办清单应用

一个简洁高效的个人待办清单Web应用，帮助您轻松管理日常任务，提高工作效率。

## ✨ 功能特点

- **添加任务**：快速添加新的待办任务，支持优先级设置
- **删除任务**：一键删除不需要的任务
- **标记完成**：轻松标记任务完成状态，完成任务自动移至底部
- **优先级管理**：支持高、中、低三级优先级设置，不同优先级用不同颜色标识
- **任务排序**：点击排序按钮，按完成状态和优先级自动排序（未完成任务在前，同组内高优先级在前）
- **搜索任务**：支持任务内容的实时搜索
- **本地存储**：自动保存任务数据到浏览器本地存储，刷新页面数据不丢失

## 🎯 使用说明

1. 在输入框中输入任务内容
2. 从下拉菜单中选择任务优先级（高、中、低）
3. 点击"添加"按钮创建新任务
4. 使用任务右侧的"完成"按钮切换任务状态
5. 使用"删除"按钮永久移除任务
6. 使用搜索框快速查找包含特定关键词的任务
7. 点击"排序"按钮按完成状态和优先级整理任务列表

## 🔧 本地运行

1. 克隆或下载项目文件
2. 直接在浏览器中打开 `index.html` 文件
3. 开始使用您的个人待办清单！

## 📱 响应式设计

应用采用响应式设计，在桌面端和移动端均有良好的使用体验。在小屏幕设备上，界面布局会自动调整以适应触控操作。
